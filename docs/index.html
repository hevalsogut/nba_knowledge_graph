Description

This ontology models the professional NBA basketball domain as a formal OWL-based knowledge graph. It captures structural and relational knowledge about players, coaches, teams, arenas, and conferences, enabling machine-readable reasoning and SPARQL querying over sports data.

The ontology was developed using a neurosymbolic approach that combines LLM-based triple extraction from Wikipedia biographies with structured validation from the official NBA API, grounding the knowledge graph in verified factual data and minimising hallucinations.

Tools used: Protégé (modelling), RDFLib/Python (population), Meta-Llama-3-8B via Hugging Face (NLP extraction), SHACL (validation).

Competency Questions

The ontology is designed to answer the following competency questions via SPARQL:

CQ1 – Player Team & Position▶

"What is the current professional team and playing position of a specific player?"

CQ2 – Team Coach▶

"Who is the head coach responsible for a specific NBA organization?"

CQ3 – Aggregate Statistics▶

"What is the average height of players in the Western Conference?"

CQ4 – Arena & Capacity▶

"Where is the home arena of a given team and what is its seating capacity?"

SPARQL Query Examples

CQ1 – Player team and position

PREFIX nba: <http://www.semanticweb.org/veliu/ontologies/2026/3/untitled-ontology-10/> SELECT ?player ?team ?position WHERE { ?player a nba:Player ; nba:playsFor ?team ; nba:playPosition ?position . FILTER(CONTAINS(STR(?player), "Aaron_Gordon")) }

CQ2 – Head coach of a team

PREFIX nba: <http://www.semanticweb.org/veliu/ontologies/2026/3/untitled-ontology-10/> SELECT ?coach ?coachName WHERE { ?coach a nba:Coach ; nba:coaches nba:Boston_Celtics ; nba:playerName ?coachName . }

CQ3 – Average height in Western Conference

PREFIX nba: <http://www.semanticweb.org/veliu/ontologies/2026/3/untitled-ontology-10/> SELECT (AVG(?height) AS ?avgHeight) WHERE { ?player a nba:Player ; nba:hasHeight ?height ; nba:playsFor ?team . ?team nba:belongsTo nba:Western_Conference . }

CQ4 – Arena and capacity

PREFIX nba: <http://www.semanticweb.org/veliu/ontologies/2026/3/untitled-ontology-10/> SELECT ?arena ?capacity WHERE { nba:Golden_State_Warriors nba:hasHomeArena ?arena . OPTIONAL { ?arena nba:hasCapacity ?capacity } }
